vscode使用和配置

零、引言

配置vscode前须知：

  1. vscode是一个功能比较全的编辑器

  2. c/c++插件：语法高亮工具

  3. chinese插件：vscode汉化工具

  4. tasks.json：定义了程序编译规则的配置文件

  5. launch.json：定义了程序调试（debug）规则的配置文件

  6. code runner插件：编译规则插件（安装该插件就可不用配置tasks.json就可以编译了）

一、下载安装

  1. vscode下载：https://code.visualstudio.com/

  2. 打开下载页面有两个选项 “stable（稳定版本）”和“insiders（内部版本）”，一般选择stable（稳定版本）

  3. 修改一下安装路径，其他默认即可

二、vscode中文支持

  1. 打开vscode，选中最左侧的工具栏最下面的插件安装

  2. 输入chinese，选择安装

  3. 安装完成，重启vscode即可

三、c/c++编译环境

安装c/c++插件

  1. 打开vscode，选中最左侧的工具栏最下面的插件安装

  2. 输入c/c++，选择安装

  3. 安装成功即可用
    注意：安装c/c++插件只有命令补全的功能，没有编译器，编译器需要另外安装

安装Chinese插件

vscode安装完是英文的，如果想要中文就要安装一下Chinese插件

  1. 打开vscode，选中最左侧的工具栏最下面的插件安装

  2. 输入chinese，选择安装

  3. 安装成功,重启vscode即可

配置编译器

windows编译器选择mingw64

  1. 下载地址：https://sourceforge.net/projects/mingw-w64/files/

  2. 下载的文件：进入网站后不要点击 “Download Lasted Version”（这个是在线安装包），往下滑，找到最新版的 “x86_64-posix-seh”。

  3. 安装MinGW：下载后是一个7z的压缩包，解压后移动到你想安装的位置即可。我的安装位置是：D:\mingw64 （路径最好不要有中文或者空格）

  4. 配置对象：MinGW，所以把你刚刚安装MinGW的路径拷贝一下

  5. 打开cmd，输入gcc -v测试，显示版本信息说明安装成功

四、编译文件（方法一）

编译例程（单文件）

  1. 新建空文件夹Code

  2. 打开VScode –> 打开文件夹 –> 选择刚刚创建的文件夹Code

  3. 新建test.cpp文件(以最简单的 HelloWorld.cpp 为例)

  4. 点击”终端”->”配置默认生成任务…”

  5. 选择“c/c++:g++.exe 生成活动文件”（安装完mingw64vscode就能自动检测到）

  6. 文件夹会自动生成.vscode目录，和tasks.json文件（无需修改）

    tasks.json就是定义如何去编译程序的配置文件

  7. 点击“HelloWorld.cpp文件”，点击“终端”->“运行生成任务”，会生成可执行文件

  8. 点击调试框的“+”号，新建一个终端就可以执行生成文件了

编译例程（多文件）

  1. 在上面的code文件夹创建test.cpp和test.h文件,HelloWorld.cpp修改内容。内容如下

  2. 将要编译的文件添加到tasks.json的参数配置，实现支持编译多个文件

  3. 点击“HelloWorld.cpp文件”，点击“终端”->“运行生成任务”，会生成可执行文件

  4. 执行文件即可得到结果

调试例程（单、多文件）

  1. 进入调试界面创建launch.json，选择 C++(GDB/LLDB)，再选择 g++.exe

  2. 选择完环境会跳出选择配置，点击”g++.exe-生成和调试活动文件”，之后会自动生成 launch.json 配置文件

  3. 默认生成launch.json文件（无需修改）

    launch.json就是定义如何去调试程序的配置文件

    注】： launch.json 文件中 “preLaunchTask” 的值 必须与 tasks.json 文件中 “label”的值一致。值的设置看个人喜好，保持默认也是OK的。

  4. 点击”HelloWorld.cpp文件”，设置断点

  5. “运行”->”启动调试”

  6. 开始调试程序

  7. 若想要弹窗、需要配置刚才生成的”launch.json”文件

四、编译文件（方法二）

安装code runner插件

安装coderunner之后就可以不需要配置task.json文件既可编译程序

  1. 打开vscode，选中最左侧的工具栏最下面的插件安装

  2. 输入code runner，选择安装

  3. 安装成功即可用

  4. code runner安装成功在界面上便有编译按钮，后续编译点击这个按钮即可

编译文件（单文件）

  1. 新建空文件夹Code

  2. 打开VScode –> 打开文件夹 –> 选择刚刚创建的文件夹Code

  3. 新建test.cpp文件(以最简单的 HelloWorld.cpp 为例)

  4. 点击vscode编译按钮，即可编译程序
